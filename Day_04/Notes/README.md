# Day 04: Basic Flip-Flop Implementations

## Overview
This directory contains the Verilog HDL source code and corresponding testbenches for the four fundamental sequential logic memory elements: D, T, JK, and SR Flip-Flops. 

## Project Files

### D (Data) Flip-Flop
*   **`d_flipflop.v`**: The core Verilog module implementing the D Flip-Flop behavior.
*   **`tb_d_flipflop.v`**: The simulation testbench used to verify the D Flip-Flop logic.

### T (Toggle) Flip-Flop
*   **`t_flipflop.v`**: The core Verilog module implementing the T Flip-Flop behavior.
*   **`tb_t_flipflop.v`**: The simulation testbench used to verify the T Flip-Flop logic.

### JK Flip-Flop
*   **`jk_flipflop.v`**: The core Verilog module implementing the JK Flip-Flop behavior.
*   **`tb_jk_flipflop.v`**: The simulation testbench used to verify the JK Flip-Flop logic.

### SR (Set-Reset) Flip-Flop
*   **`sr_flipflop.v`**: The core Verilog module implementing the SR Flip-Flop behavior.
*   **`tb_sr_flipflop.v`**: The simulation testbench used to verify the SR Flip-Flop logic.
