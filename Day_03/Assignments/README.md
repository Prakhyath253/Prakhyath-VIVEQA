# Assignment – Combinational Circuits

This folder contains the assignment question paper and handwritten solutions for the **Combinational Circuits** topic in Digital Electronics.

## Files
- D3-Assignment_Questions.pdf
- D3-Assignment_Solutions.pdf

## Description

This assignment covers the design and implementation of combinational logic circuits using multiplexers and Boolean simplification techniques. The topics include:

- Implementing Boolean functions using **2:1 Multiplexers (MUX)** and **NOT gates**.
- Implementing Boolean functions using **4:1 Multiplexers (MUX)**.
- Designing a **Full Adder** using only **2:1 Multiplexers**.
- Simplifying Boolean expressions using **Karnaugh Maps (K-Maps)** with don't-care conditions.
- Implementing the simplified **Sum of Products (SOP)** expression using a **4:1 Multiplexer**.

**D3-Assignment_Questions.pdf** contains the original assignment questions, while **D3-Assignment_Solutions.pdf** contains handwritten, step-by-step solutions for all the problems.
