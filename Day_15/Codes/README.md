# Day 15 - 4-Bit Asynchronous Up Counter

This folder contains the Verilog HDL programs for implementing a **4-Bit Asynchronous Up Counter** using T Flip-Flops on the FPGA development board.

## Files

- `async_counter.v`
- `async_counter_tb.v`
- `async_counter.xdc`

## Description

`async_counter.v` contains the Verilog implementation of a 4-bit asynchronous (ripple) up counter using T Flip-Flops. The first flip-flop is driven by the system clock, while each subsequent flip-flop is triggered by the complemented output of the previous stage, causing the counter to increment asynchronously. An active-high reset initializes all flip-flops to zero.

`async_counter_tb.v` contains the testbench used to simulate and verify the operation of the 4-bit asynchronous up counter. The simulation applies a reset signal and observes the binary counting sequence generated by the ripple counter on successive clock pulses.

`async_counter.xdc` contains the Xilinx Design Constraints (XDC) file that maps the system clock, reset button, and the four counter output bits to the onboard LEDs. It also specifies the **LVCMOS33** I/O standard required for FPGA implementation.
