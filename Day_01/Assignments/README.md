# Assignment – Number Systems

This folder contains the assignment question paper and handwritten solutions for the **Number Systems** topic in Digital Electronics.

## Files
- `D1-Assignment_Questions.pdf`
- `D1-Assignment_Solutions.pdf`

## Description
This assignment covers the fundamental concepts of **Number Systems** used in Digital Electronics. The topics include:

- Binary, Decimal, Octal, and Hexadecimal number system conversions.
- Base (radix) conversions.
- Determination of an unknown radix.
- Arithmetic operations in different number systems.
- Binary to Gray Code and Gray Code to Binary conversions.

`D1-Assignment_Questions.pdf` contains the original assignment questions, while `D1-Assignment_Solutions.pdf` contains handwritten, step-by-step solutions for all the problems.
