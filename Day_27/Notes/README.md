# Day 27 - Binary to Gray Code Converter and Gray Code to Binary Converter

## Introduction

This project demonstrates the implementation of a Binary to Gray Code Converter and Gray Code to Binary Converter using Verilog HDL. The design performs both conversions using combinational logic without requiring a clock signal. Binary values are converted into Gray Code using XOR operations while Gray Code values are converted back into Binary using successive XOR operations.

This project introduces code conversion combinational logic XOR operations digital encoding digital decoding and FPGA implementation using Verilog HDL.

---

## Binary to Gray Code Conversion

Gray Code is a binary numbering system in which only one bit changes between consecutive numbers.

```verilog
assign binary_to_gray = binary ^ (binary >> 1);
```

Each Gray Code bit is generated by performing an XOR operation between adjacent Binary bits.

---

## Gray Code to Binary Conversion

Binary values are recovered from Gray Code using successive XOR operations.

```verilog
assign gray_to_binary[3] = gray[3];
assign gray_to_binary[2] = gray_to_binary[3] ^ gray[2];
assign gray_to_binary[1] = gray_to_binary[2] ^ gray[1];
assign gray_to_binary[0] = gray_to_binary[1] ^ gray[0];
```

The most significant bit remains unchanged while each remaining Binary bit is obtained using XOR operations.

---

## Input and Output Ports

```verilog
input [3:0] binary;
input [3:0] gray;

output [3:0] binary_to_gray;
output [3:0] gray_to_binary;
```

**binary** – 4-bit Binary input

**gray** – 4-bit Gray Code input

**binary_to_gray** – Converted Gray Code output

**gray_to_binary** – Converted Binary output

---

## Working Principle

Initially the Binary input is converted into Gray Code.

The Gray input is simultaneously converted into Binary.

Both conversions are performed continuously using combinational logic.

The outputs update immediately whenever the inputs change.

---

## Combinational Logic

The converter is a combinational circuit.

The operation depends on:

- Binary Input
- Gray Code Input
- XOR Logic

No clock or memory elements are required.
