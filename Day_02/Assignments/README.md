# Assignment – Boolean Algebra

This folder contains the assignment question paper and handwritten solutions for the Boolean Algebra topic in Digital Electronics.

## Files

- D2-Assignment_Questions.pdf
- D2-Assignment_Solutions.pdf

## Description

This assignment covers the fundamental concepts of Boolean Algebra used in Digital Electronics. The topics include:

- Deriving Boolean expressions from logic circuits.
- Simplification of Boolean expressions using Boolean postulates and theorems.
- Identifying the equivalent single gate for a given logic circuit.
- Minimization of Boolean expressions using Karnaugh Maps (K-Maps).

`D2-Assignment_Questions.pdf` contains the original assignment questions, while `D2-Assignment_Solutions.pdf` contains handwritten, step-by-step solutions for all the problems.
